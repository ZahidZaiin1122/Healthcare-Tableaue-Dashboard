# Healthcare Patient Satisfaction Dashboard 📊

## Overview
An interactive Tableau dashboard analyzing healthcare 
patient satisfaction using 9,216 patient records across 
multiple departments, gender, and race categories.

## Live Dashboard
🔗 [View on Tableau Public](https://public.tableau.com/app/profile/zahid.ahmad1692/viz/HealthcarePatientsatifactionDashboard/Dashboard1)

## Key Findings
- Cardiology has highest average patient wait time
- Patient satisfaction scores similar across genders
- Native American/Alaska Native patients report 
  highest satisfaction
- General Practice shows highest satisfaction by department

## Dashboard Features
- 4 Interactive Charts
- Satisfaction by Gender
- Wait Time by Department
- Satisfaction by Department
- Satisfaction by Race

## Tools Used
- Tableau Public 2025
- Healthcare Analytics Patient Flow Dataset (Kaggle)
- 9,216 patient records
